Requirements

  • Windows 10-11
  • Python 3.13+
  • PIP modules: pillow, geopy, pythonnet
    Ensure you have PIP installed. Download modules by entering "pip install pillow geopy pythonnet" into CMD

Installation Steps

  1. Download the newest version and extract
  2. Right click RouteProcessor.dll, select properties and check the box to unblock
    This dll file contains the C# function for reading your route data. Doing this tells Windows to trust it
  3. Run _routelog.bat to launch the program
  4. Select your aircraft types using function 9
  5. Import official routes through Data Settings (function 13)
  6. Set your home FIR using function 10
